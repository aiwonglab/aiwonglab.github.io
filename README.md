# AI Wong Lab

A. Ian Wong, MD, PhD

## AI Wong Lab Website

Professional research website for the AI Wong Lab, focusing on critical and acute care informatics.

### Features

- 29+ peer-reviewed publications (2021-2025)
- Extensible team management system
- Google Scholar BibTeX import tools
- Responsive design optimized for all devices

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

### Updating Content

- **Publications**: Edit `src/pages/publications.astro` or use `scripts/import-publications.js`
- **Team**: Edit `src/data/team.ts`
- **Contact**: Edit `src/pages/contact.astro`

For more details, see documentation in `src/data/README.md` and `scripts/README.md`.
