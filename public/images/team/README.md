# Team Images

This directory contains profile images for team members.

## Naming Convention

All images should follow the format: **`lastName_firstName.jpg`**

Examples:
- `wong_ian.jpg` - A. Ian Wong
- `alwakeel_mahmoud.jpg` - Mahmoud Alwakeel
- `zhang_brenda.jpg` - Brenda Zhang

## Image Specifications

- **Format**: JPG, PNG, or SVG
- **Recommended size**: 400x400 pixels (square)
- **File size**: Keep under 500KB for web performance
- **Background**: Solid color or professional background

## Placeholder Images

The SVG files in this directory are placeholders generated automatically. Replace them with actual photos by:
1. Adding your photo to this directory with the correct filename
2. Ensuring it follows the naming convention above
3. The website will automatically use your photo

## Updating Team Data

When adding a new team member, update `/src/data/team.ts` with their information and add their image to this directory following the naming convention.
